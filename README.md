# VeloLab

Website completo em React + Vite para uma bicicletaria premium fictícia. O projeto prioriza UI/UX, responsividade, animações fluidas, SEO básico e interações JavaScript.

## Stack

- React + Vite
- React Router DOM
- Framer Motion
- React Helmet Async
- Lucide React
- Configurador 2D interativo em SVG/React (fallback performático ao 3D)
- CSS puro organizado por componentes e estilos globais

## Páginas

- Home (`/`)
- Bicicletas (`/bicicletas`)
- Oficina/Serviços (`/oficina`)
- Experiência/Performance (`/performance`)
- Contato (`/contato`)

## Interações implementadas

- Menu mobile animado
- Navbar com estado visual ao scroll
- Filtro interativo de bicicletas
- Animações on-scroll com Framer Motion
- Seção técnica de bicicleta reagindo ao scroll
- Configurador visual com troca de cor, pneu, guidão e acabamento
- FAQ accordion
- Formulário de contato com validação JavaScript, erros, loading fake e sucesso
